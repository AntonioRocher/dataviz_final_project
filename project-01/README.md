## Mini-Project 1
# Traffic Data Visualization Project

This project explores how traffic volume changes based on time of day, weekday, and weather conditions. The goal is to identify meaningful patterns in traffic behavior and apply data visualization techniques.

---

## Project Overview

The analysis focuses on three main questions:
- How does traffic change throughout the day?
- Are there differences in traffic between weekdays and weekends?
- How do weather conditions affect traffic volume?

---

## Visualizations

### Graph 1: Traffic Volume by Hour
This visualization shows how traffic varies across a 24-hour period. It reveals clear commuting patterns, with traffic increasing in the morning, peaking in the evening, and decreasing overnight.

---

### Graph 2: Traffic Volume by Weekday (Interactive)
This is an interactive bar chart that compares average traffic across weekdays.

The interactivity allows users to hover over bars to view exact values and explore differences between days more clearly than a static chart.

---

### Graph 3: Traffic Volume by Weather Condition
This chart compares traffic volume across different weather conditions. It highlights how more severe weather tends to reduce traffic levels.

---

## Key Findings

- Traffic follows predictable daily commuting patterns  
- Weekdays have significantly higher traffic than weekends  
- Weather conditions can influence traffic volume  

---

## Accessibility

All visualizations were designed with accessibility in mind:
- Colorblind-safe color choices were used where applicable  
- Clear labels and minimal themes improve readability  
- One visualization includes interactivity for deeper exploration  
- Alternative text descriptions are included in the source document for screen readers  

---

## Conclusion

This project demonstrates how traffic patterns can be effectively analyzed using data visualization. Combining static and interactive charts provides both overview insights and deeper exploration of the data.
